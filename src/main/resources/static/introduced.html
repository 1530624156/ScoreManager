成绩管理系统-系统介绍

系统功能：

  1. 学生管理：记录每个学生的信息
  2. 课程管理：课程信息管理
  3. 成绩管理：记录每名学生的每门课成的成绩信息
  4. 管理员：管理员可以修改管理员信息

技术选型：

  1. 开发语言：Java/HTML/CSS/JavaScript
  2. 后端框架：springBoot
  3. 前端框架：Layui,jquery
  4. 数据库： Mysql
  5. 持久层框架：MybatisPlus
  6. 工具类：Fastjson,log4j,poi,MyJdbc,MyDate,lombok
  7. 更多……
